# mk-zh-lab

本仓库用于归档和发布多个独立 HTML 页面。这些页面通常为实验性、功能性或展示性的前端内容，不针对单一主题或服务。

## 目录结构

```
.
├── README.md          # 本文件
├── index.html         # 仓库入口页面（可选）
└── pages/             # 存放各独立 HTML 页面
    ├── example1.html
    ├── example2.html
    └── ...
```

- `pages/`：主目录，每个 HTML 文件为一个独立页面。
- 根目录下的 `index.html` 可作为导航或简介页（如有）。

## 页面归档说明

每个 HTML 页面均为自包含实现，依赖资源（如 CSS、JavaScript）尽量内联或引用公开 CDN。页面内容可能包含：

- 前端小工具或演示
- 静态信息展示
- 实验性交互设计

**注意**：页面内容仅代表归档时的状态，可能不会持续更新。部分页面可能依赖外部资源，若外部资源失效，页面可能无法正常显示。

## 维护说明

- 本仓库以归档和展示为主要目的，不接受针对具体页面的功能请求或定制。
- 如需引用或修改页面内容，请遵守相关开源协议（如适用）。
- 新增页面时，建议在 `pages/` 下创建独立文件，并保持文件命名清晰简洁。

## 其他

- 页面可通过 GitHub Pages 直接访问（如已启用）。
- 仓库不保证所有页面长期可用或兼容所有浏览器。

如有问题，欢迎提交 Issue 讨论。
